Task 10​

在 C:\Users\ 目录下还剩下多少个文件？（递归计算）

首先，使用筛选功能，对 ParentPath 列的数据进行筛选，使用菜单中的文本筛选功能

然后将结果复制黏贴为新的工作表，将 IsDirectory 筛选为 FALSE 模式，继续筛选
