## 4

> 黑客获取的权限 flag\{whoami 后的值 \}

直接执行 udf 中的函数即可

```sql
MariaDB [(none)]> select * from mysql.func;
+----------+-----+-------------+----------+
| name     | ret | dl          | type     |
+----------+-----+-------------+----------+
| sys_eval |   0 | mysqludf.so | function |
+----------+-----+-------------+----------+
1 row in set (0.00 sec)

MariaDB [(none)]> select sys_eval("whoami");
+--------------------+
| sys_eval("whoami") |
+--------------------+
| mysql              |
+--------------------+
1 row in set (0.01 sec)
```

即可得到答案

flag{mysql}
