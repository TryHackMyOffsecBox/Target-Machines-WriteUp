# Azure Penetration Testing

## Introduction

More than 95 percent of Fortune 500 companies use Azure, making it one of the most critical cloud platforms today. Entra ID (formerly Azure AD) is among the largest web-based identity providers, and the ability to assess and secure Azure environments is a highly sought-after skill.

At Altered Security, we specialize in hands-on enterprise security training and have trained over 7,000 security professionals in Enterprise and Azure security. Recognizing the growing need for structured and high-quality learning material, we present "Introduction to Azure Penetration Testing."

This course is designed for security professionals, penetration testers, and those looking to deepen their expertise in Azure security. Through structured content and practical exercises, participants will gain a solid understanding of key attack techniques, security misconfigurations, and defense strategies in Azure environments.
