1. 结合邮件正文发掘 flag​

在用户桌面发现存在邮件文件

对其进行分析，为 zip 压缩文件

查看eml邮件文件

解压附件中的压缩包，即可得到flag.txt

flag{Let-'s-Begain#this@Game}
